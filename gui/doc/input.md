A control that displays some text that can potentially be edited.

## Usage

### Get user input

You can create an input field bound to a variable with the following content:

!!! example "Page content"

    === "Markdown"

        ```
        <|{value}|input|>
        ```
  
    === "HTML"

        ```html
        <taipy:input>{value}</taipy:input>
        ```
