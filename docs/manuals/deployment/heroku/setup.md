# Heroku setup

## 1. Create an account on Heroku

Go on [Heroku Signup](https://signup.heroku.com/) to create an account on Heroku.

## 2. Install the Heroku CLI

To run commands that will provision your infrastructure, you should install the Heroku CLI.
The complete documentation is available on the [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli) page.
